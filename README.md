# 🚀 Features
- Create tasks
- Create subtasks
- Create workspaces

# 🏁 Get Started
### Install postgres or use docker

```docker-compose up```

### Install packages

```go mod download or make install```

### Run project main.go file

```go run cmd/main.go or make run```
